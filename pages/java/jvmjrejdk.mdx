# JRE(Java Runtime Environment)
- 자바 실행환경으로 자바 프로그램을 실행시키는데만 사용되는 패키지
- JVM, Java Class Library, Java Command, 기타 등등 포함


# JDK(Java Development Kit)
- 자바 개발키트의 약어, Java의 SDK 라고 생각하면된다.
- JRE, 컴파일러, 디버거, 디버거, javap, jar(명령어)

## SDK(Software Development Kit)란
- 소프트웨어 개발 키트로 특정 소프트웨어를 만들기 위한 도구 모음
- 어플리케이션을 생성하거나 하나의 패키지로 많은 기능을 추가하는데 사용
