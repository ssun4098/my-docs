# Spring-Data-Redis
- 스프링에서 지원해주는 Redis 라이브러리
- Jedis, Lettuce 등 포함되어 있다.
- RedisTempate과 RedisRepository 방식을 사용할 수 있다.

## Lettuce 라이브러리와 커넥션 설정
스프링에서는 기본적으로 Lettuce를 사용한다. Lettuce는 내부에 Netty를 사용하고 있어 비동기 논블로킹으로 구현되어 있다. 

## RedisConnectionFactory
- RedisTempate은 명령어를 실행하고 RedisConnectionFactory에 커넥션 관리를 위임한다.
- 구현체로는 JedisConnectionFactory와 LettuceConnectionFactory 클래스를 제공한다.

## RedisConfiguration
Redis 설정 정보를 포함하는 인터페이스이며 아키텍처에 따른 구현체를 제공한다.
- RedisStandaloneConfiguration: 단독으로 구성된 레디스 서버에 커넥션을 맺을 때 사용
- RedisStaticMasterReplicationConfiguration: 레디스 마스터 레플리카 구조로 구성된 레디스 서버에 커넥션을 맺을 때 사용
- RedisSocketConfiguration: 유닉스 도메인 소켓을 이용하여 로컬 호스트에 설치된 레디스 서버에 커넥션을 맺을 때 사용
- RedisSentinelConfiguration: 레드스 센티넬 구조로 구성된 레디스 센티넬 서버에 커넥션을 맺을 때 사용
- RedisClusterConfiguration: 레디스 클러스터 구조로 구성된 레디스 서버에 커넥션을 맺을 때 사용
