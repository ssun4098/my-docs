# 영속성 전이(Transitive Persistence)
- 특정 엔티티를 영속 상태로 만들 때 연관엔 엔티티도 함께 영속 상태로 만듬
- 영속성 전이를 사용 및 관리하고 싶다면 CASCADE를 사용하면 된다.

# CascadeTpe
```
public enum CascadeType { 
    ALL, // 모든 상태 변화(PERSIST, MERGE, REMOVE, REFRESH, DETACH)가 전파됨
    PERSIST, // 부모 엔티티가 영속될 때, 관련된 자식 엔티티도 영속 상태로 전환됨.
    MERGE, // 부모 엔티티가 병합될 때, 관련된 자식 엔티티도 병합됨.
    REMOVE, // 부모 엔티티가 삭제될 때, 관련된 자식 엔티티도 삭제됨.
    REFRESH, // 부모 엔티티가 새로 고침될 때, 관련된 자식 엔티티도 새로 고침됨.
    DETACH // 부모 엔티티가 준영속 상태로 변경될 때, 관련된 자식 엔티티도 준영속 상태로 전환됨.
}
```

# 고아 객체(ORPHAN)
- 부모 엔티티의 컬렉션에서 자식 엔티티와의 연관관계가 끊어진 자식 엔티티를 자동으로 삭제하는 기능
- @OneToMany or @OneToOne에서 orphanRemoval = true로 설정하면 된다.
