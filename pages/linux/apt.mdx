# APT(Advanced Packaging Tool)
- Debian기반의 Linux 서버의 소프트웨어 패키지 관리 도구
- 인터넷을 통하여 필요한 파일을 저장소에서 자동으로 모두 다운로드 해서 설치하는 방식(저장소 URL은 /etc/apt/sources.list 파일)

# dpkg vs apt
- dpkg는 패키지를 설치하면 해당 패키지만 설치하고 의존하고 있는 패키지는 설치하지 않아 불편한 점이 있다.
- apt는 deb 파일 저장소애서 의존하고 있는 패키지까지 찾아 설치해준다.

# 기본 사용법
- "apt -y install 패키지명": -y는 사용자의 확인을 모두 yes로 간주
- "apt update": 패캐지 목록 업데이트
- "apt remove/purge 패키지 이름": 패키지 삭제

# apt-cache
패키지를 설치하기 전에 패키지에 대한 정보나 의존성 문제를 미리 확인
- "apt-cache show 패키지이름": 패키지 정보 보기

# 우분투 패키지 저장소
- main: 우분투에서 공식적으로 지원하는 무료 SW
- universe: 우분투에서 지원하지 않는 무료 SW
- restricted: 우분투에서 공식적으로 지원하는 유로 SW
- multiverse: 우분투에서 지원하지 않는 유로 SW
